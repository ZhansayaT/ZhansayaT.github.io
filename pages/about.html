Website

This is my personal website. It was depeloped as the part of the "CSCI111 - Web Programming and Problem Solving" assesments. My goal was to create my personal website as an online CV using HTML, CSS and JavaScript. I followed all the technical requirements from this document.

Website consists of 2 pages:

  • CV
  • About

CV consists of all the stuff CV usually consists of:

  • Education
  • Work experience
  • Achievements
  • Personal and professional skills
  • Extracurricular activities
  • Links to the social media or Github accounts
  • Photos

I followed mobile first approach to style the website. It made everything easier, because there were no need to rewrite everything for the bigger displas, only the elements that needed to look different.

Big problem was to make all the images croped but still responsive. Luckily someone from stackoverflow faced the same problem, and I adopted thier solution

But the biggest problem was to info section always follow the user. I achieved it by using window.addEventListener("scroll") function to always check if the user scrolling down or not.
